HERO IMAGE INSTRUCTIONS
========================

To use your custom chauffeur and Escalade image:

1. Save your image as "hero-chauffeur-escalade.jpg" in this assets/images/ folder

2. The image should be:
   - High resolution (1920x1080px or larger)
   - Professional quality
   - Shows the chauffeur and black Escalade with private jet

3. The website is already configured to use: assets/images/hero-chauffeur-escalade.jpg

4. Once saved, refresh your website to see the new image

OPTIMAL IMAGE SPECS:
- Format: JPG or PNG
- Resolution: 1920x1080px minimum (2560x1440px recommended)
- File size: Under 2MB for fast loading
- Aspect ratio: 16:9 for best fit
