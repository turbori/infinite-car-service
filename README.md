# Infinite Car Service Website

A premium luxury car service website for Long Island's #1 trusted transportation company, serving customers since 2005.

## 🚗 About

Infinite Car Service provides luxury transportation between Long Island, Manhattan, and NYC airports. This website showcases our premium fleet, services, and professional chauffeurs while offering seamless booking through our integrated quote system.

## ✨ Features

- **Responsive Design**: Fully responsive across all devices and screen sizes
- **Luxury Fleet Showcase**: Detailed vehicle information with capacity and amenities
- **Service Portfolio**: Comprehensive service offerings from airport transfers to special events
- **Professional Chauffeurs**: Information about our trained and suited drivers
- **Child Safety**: Detailed child seat options and NY state regulations
- **Airport Meeting Points**: Specific pickup locations for JFK, LGA, and ISP airports
- **Quote Integration**: Direct integration with Moovs booking platform
- **Premium Animations**: Subtle welcome animations for enhanced user experience
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Tailwind CSS, DaisyUI
- **Icons**: Font Awesome 6
- **Fonts**: Google Fonts (Inter)
- **Animations**: CSS Keyframes
- **Architecture**: Static website with external booking integration
